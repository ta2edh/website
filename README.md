# TA2EDH - Personal Website

A modern, responsive personal website built with Next.js, React, and Tailwind CSS.

![Website Screenshot](https://i.imgur.com/CqgKmYL.png)

## Features

- 🎨 Modern and clean design
- 📱 Fully responsive layout
- 🌙 Dark/Light mode support
- 📝 Contact form with validation
- 🔗 Social media integration
- ⚡ Fast performance with Next.js
- 🎯 SEO optimized

## Getting Started

### Prerequisites

- Node.js 18.17 or later
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/ta2edh/website
   cd website
   ```

2. Install dependencies:
   ```bash
   npm install
   # or
   yarn install
   ```

3. Run the development server:
   ```bash
   npm run dev
   # or
   yarn dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

```
ta2edh.com/
├── app/                 # Next.js app directory
│   ├── layout.tsx      # Root layout
│   └── page.tsx        # Home page
├── components/         # React components
│   ├── ui/            # UI components
│   ├── contact-form.tsx
│   ├── header.tsx
│   ├── social-links.tsx
│   └── signal-meter.tsx
├── lib/               # Utility functions
├── public/           # Static assets
└── styles/          # Global styles
```

## Development

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run start` - Start production server
- `npm run lint` - Run ESLint

## Deployment

This website is configured for easy deployment on Vercel:

1. Push your code to GitHub
2. Import your repository on Vercel
3. Deploy!
